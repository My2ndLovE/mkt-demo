## What You'll Get

A web-based lottery system for managed agents on 4D, 3D, 5D, 6D betting operations.

---

## 8-Week Development Plan

### Week 1-2: User System (RM 4,300)
- Login system with secure passwords
- Three user roles: Agent, Moderator, Admin
- Create and manage user accounts
- Basic app layout and navigation

### Week 3: Agent Management (RM 2,150)
- Multi-level agent hierarchy (unlimited levels)
- Set commission percentages per agent
- Set weekly betting limits
- Manage upper/lower agent relationships

### Week 4: Betting - 4D (RM 2,150)
- Place 4D lottery bets
- Number entry with keypad
- Select lottery companies (M, P, T, S, B, K, W, H, E)
- Choose bet types (Big, Small, etc.)
- Weekly limit checking

### Week 5: Betting - All Games (RM 2,150)
- Support 3D, 5D, 6D games
- Batch entry (multiple bets at once)
- Quick entry shortcuts (#123, 1234*, etc.)
- View and cancel pending bets

### Week 6: Results & Commission (RM 2,150)
- Auto-sync with official lottery results
- Calculate wins/losses automatically
- Commission distribution across all levels
- Performance tracking

### Week 7: Search & Reports (RM 2,150)
- Search bets by date, number, receipt
- Live betting statistics
- Order summaries
- Export/print receipts

### Week 8: Final Reports & Testing (RM 2,150)
- Calendar view with results
- 7-day performance reports
- Winning bets summary
- Final testing and bug fixes
